# Системные диаграммы последовательностей

## __1. Ввод имени__

![Uploading Снимок экрана 2024-03-24 в 13.53.14](https://github.com/srapchegs/rtippo/assets/160636893/fcaf1385-3226-4201-8982-d8a17c8c5b10)


1. Запустить игру

| Действие          | Запустить игру                       |
|-------------------|---------------------------------------|
| Ссылки            | Прецеденты: Начать игру               |
| Предусловия       | Нет                                   |
| Постусловие       | Игра запущена, игроки добавлены, игра готова к началу раунда |

2. Игрок вводит имя

| Действие          | Игрок вводит имя                    |
|-------------------|--------------------------------------|
| Ссылки            | Прецеденты: Запустить игру          |
| Предусловия       | Игра запущена                        |
| Постусловие       | Имя игрока сохранено                 |

3. Запросить ввод следующего игрока

| Действие          | Запросить ввод следующего игрока            |
|-------------------|----------------------------------------------|
| Ссылки            | Прецеденты: Завершить добавление игроков    |
| Предусловия       | Игра запущена, все игроки добавлены         |
| Постусловие       | Ожидание ввода следующего игрока            |

4. Завершить добавление игроков

| Действие          | Завершить добавление игроков            |
|-------------------|-----------------------------------------|
| Ссылки            | Прецеденты: Игрок вводит имя            |
| Предусловия       | Игра запущена, все игроки добавлены     |
| Постусловие       | Все игроки добавлены, игра готова к началу раунда |
